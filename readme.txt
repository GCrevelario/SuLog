============================Database setup==========================
--> On conn.js file replace the field ### with your MySQL password.

--> Use the follow query to create the database:

CREATE DATABASE sulog

--> The tables will be created automatically.

============================Aplication setup=========================
--> Use the follow command to install the npm dependencies:
NPM INSTALL

--> After the step abouve, issue the follow command to start the application:
NPM START
